1. Introduction to Vireo II

Vireo II is a rewrite of the Vireo I kernel, which was very unstable and poorly planned. Vireo I lacked a good architecture and good coding practices. The new Vireo II should improve this quite a bit. It's rewritten, though not entirely from scratch: some code from the Vireo I kernel will be upcycled in the new Vireo II kernel.
